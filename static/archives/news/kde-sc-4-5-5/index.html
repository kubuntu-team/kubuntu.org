KDE Releases 4.5.5

Packages for the release of the KDE Software Compilation 4.5.5 are available for Kubuntu 10.10.

Bugs in packaging should be reported to kubuntu-ppa on Launchpad. Bugs in the software to KDE.

Users of 10.10 can install it from the Kubuntu Updates PPA.
To update, use the Software Repository Guide to add the following repository to your software sources list:

ppa:kubuntu-ppa

Thanks to Steve for his help with this release.
