KDE SC 4.3 – Kubuntu https://kubuntu.org Friendly Computing Wed, 25 Mar 2015 16:37:47 +0000 en-US hourly 1 https://wordpress.org/?v=5.8.3 KDE Software Compilation 4.3.5 in Backports https://kubuntu.org/news/kde-sc-4-3-5/ Thu, 28 Jan 2010 16:37:52 +0000 KDE SC 4.3.5 is now in karmic-backports.

The Backports repository is not enabled by default, so you may need to enable it in KPackageKit. Instructions for this are found on the help wiki.

To manually enable it, edit your /etc/apt/sources.list and uncomment the line
deb http://archive.ubuntu.com/ubuntu/ karmic-backports main restricted
universe multiverse.

When upgrading from 4.3.2 to 4.3.4, some package recommendations have changed, so using dist-upgrade (or full-upgrade in aptitude) may be needed.

Thanks to Lex for preparing these packages.

]]> KDE SC 4.3.4 moved from PPA to karmic-backports https://kubuntu.org/news/kde-sc-434-moved-ppa-karmic-backports/ Tue, 05 Jan 2010 14:55:46 +0000 KDE SC 4.3.4 is now in karmic-backports and will be removed from the PPA shortly.
Even if you have the PPA packages, please upgrade to the versions in
karmic-backports. There are security fixes included in karmic-backports
that were not in the PPA packages.

The Backports repository is not enabled by default, so you may need to enable it in KPackageKit. Instructions for this are found here.

To manually enable it, edit your /etc/apt/sources/list and uncomment the line
deb http://archive.ubuntu.com/ubuntu/ karmic-backports main restricted
universe multiverse.

Packages for i386, amd64, and powerpc have finished building. When upgrading
from 4.3.2 to 4.3.4, some package recommendations have changed, so using
dist-upgrade (or full-upgrade in aptitude) may be needed.

]]> KDE 4.3 released https://kubuntu.org/news/kde-4-3-released/ Tue, 04 Aug 2009 18:04:42 +0000 KDE 4.3 packages are now available.

To see what’s new, see the release announcement

Users of our latest development release, Karmic, can upgrade to the latest packages. Note: The last few parts are still building and will be available in a few hours.

Users of our stable 9.04 release can install it from the Kubuntu Backports PPA.

deb http://ppa.launchpad.net/kubuntu-ppa/backports/ubuntu jaunty main

It is strongly recommended that you verify the integrity of these packages by installing the archive’s GPG key. You may do this by following the instructions on this page.

This archive includes a new network manager application, run “knetworkmanager” (the Plasmoid is just a dummy currently).

Packages for Jaunty are not officially supported. KDE 4.3 will be part of Kubuntu Karmic 9.10 to be officially released this October.
