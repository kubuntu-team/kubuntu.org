]]> KOffice 2.1 Available https://kubuntu.org/news/koffice-2-1-0/ Tue, 24 Nov 2009 21:42:14 +0000 KOffice 2.1 has been released. This release is a marked improvement of almost all parts of KOffice compared to 2.0. Version 2.0 was labeled a “platform release”, which was meant as a first preview of the framework and new UI paradigm. In version 2.1, most applications and components have improved significantly but should still only be used by early adopters and probably not as the primary worktool.

Users of our latest development release, Lucid, can upgrade to the latest packages.

Users of our stable 9.10 release can install koffice-kde4 from the Kubuntu Backports PPA.

deb http://ppa.launchpad.net/kubuntu-ppa/backports/ubuntu karmic main

It is strongly recommended that you verify the integrity of these packages by installing the archive’s GPG key. You may do this by following the instructions on this page.
