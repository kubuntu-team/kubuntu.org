KDE SC 4.5.1 – Kubuntu https://kubuntu.org Friendly Computing Wed, 25 Mar 2015 21:10:21 +0000 en-US hourly 1 https://wordpress.org/?v=5.8.3 Packages Available for KDE Platform, Plasma and Applications 4.5.1 https://kubuntu.org/news/packages-available-kde-platform-plasma-and-applications-451/ Fri, 03 Sep 2010 03:13:07 +0000 Packages for the release of the KDE Software Compilation 4.5.1 are available.

Bugs in packaging should be reported to kubuntu-ppa on Launchpad. Bugs in the software to KDE.

Users of 10.04 LTS can install it from the Kubuntu Backports PPA.
To update, use the Software Repository Guide to add the following repository to your software sources list:

ppa:kubuntu-ppa/backports

Packages are currently building for our Maverick development release and should become available in the coming hours.

Thanks to all the ninjas for packaging this release.
