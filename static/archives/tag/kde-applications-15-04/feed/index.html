KDE Applications 15.04 – Kubuntu https://kubuntu.org Friendly Computing Sun, 19 Jul 2015 00:37:36 +0000 en-US hourly 1 https://wordpress.org/?v=5.8.3 KDE Applications 15.04.1 Available for Kubuntu 15.04 https://kubuntu.org/news/kde-applications-15041-available-kubuntu-1504/ Sat, 30 May 2015 00:30:35 +0000 https://www-new.kubuntu.org/?p=2332

Packages for the release of KDE Applications 15.04.1 are available for Kubuntu 15.04. You can get it from the Kubuntu Backports PPA.

Bugs in the packaging should be reported to kubuntu-ppa on Launchpad. Bugs in the software to KDE.

To update, use the Software Repository Guide to add the following repository to your software sources list:

ppa:kubuntu-ppa/backports

Special thanks to sgclark for packaging this update and to the testers who hang out on the #kubuntu-devel IRC channel.
