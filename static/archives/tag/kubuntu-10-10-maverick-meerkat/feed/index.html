]]> Maverick Alpha 3 Out https://kubuntu.org/news/maverick-alpha-3-out/ Thu, 05 Aug 2010 15:24:48 +0000 The third alpha of Maverick Meerkat has been released. This contains a release candidate version of KDE Platform and Plasma 4.5.

Other highlights of Alpha 3 include :

  • Dragon continues on as the default video player
  • Message Indicator support is enabled by default for Kopete and Quassel
  • Qt was updated to the current 4.7 beta release

This release contains a couple of notable bugs. KDM times out when running from a CD. The installer is also not shown on the desktop.
