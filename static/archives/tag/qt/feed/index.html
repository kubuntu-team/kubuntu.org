Qt – Kubuntu https://kubuntu.org Friendly Computing Wed, 25 Mar 2015 21:11:58 +0000 en-US hourly 1 https://wordpress.org/?v=5.8.3 QtCreator 2.1 Beta 1 Packaged https://kubuntu.org/news/qtcreator-21-beta-1-packaged/ Sat, 09 Oct 2010 18:00:17 +0000 Packages for the release of the QtCreator 2.1 Beta 1 are now available for Kubuntu 10.10.

Bugs in packaging should be reported to kubuntu-ppa on Launchpad. Bugs in the software to QtCreator.

Users of 10.10 Release Candidate can install it from the Kubuntu Beta Backports PPA.
To update, use the Software Repository Guide to add the following repository to your software sources list:

ppa:kubuntu-ppa/beta

There are no packages planned for Kubuntu 10.04 LTS, we recommend updating to 10.10 if you want to try the newest software, either upgrade to the release candidate or wait for the final release on Sunday.

Thanks to Felix Geyer for helping to package this release.
