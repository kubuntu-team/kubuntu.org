]]> KDE SC 4.4 Release Candidate 1 packages available https://kubuntu.org/news/kde-sc-4-4-rc-1/ Sat, 09 Jan 2010 18:24:30 +0000 Packages for the first release candidate of the KDE Software Compilation 4.4 are available.

]]> Christmas Bonus: KDE SC, Amarok, KDevelop Betas https://kubuntu.org/news/kde-sc-4-4-beta-2/ Tue, 22 Dec 2009 17:20:44 +0000 The second beta of KDE SC 4.4 has packages available in our Beta PPA. There are also betas of Amarok and KDevelop available.

Users of our latest development release, Lucid, can upgrade to the latest packages.

Users of our stable 9.10 release can install it from the Kubuntu Beta PPA.

deb http://ppa.launchpad.net/kubuntu-ppa/beta/ubuntu karmic main

Then do sudo apt-get update then sudo apt-get dist-upgrade and sudo apt-get install kubuntu-desktop.

It is strongly recommended that you verify the integrity of these packages by installing the archive’s GPG key. You may do this by following the instructions on this page.

Many thanks to Lex for his hard work on these packages.
