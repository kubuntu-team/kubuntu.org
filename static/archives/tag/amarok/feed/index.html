Amarok – Kubuntu https://kubuntu.org Friendly Computing Tue, 24 Mar 2015 03:55:06 +0000 en-US hourly 1 https://wordpress.org/?v=5.8.3 Amarok 2.8 https://kubuntu.org/news/amarok-2-8/ Fri, 16 Aug 2013 15:32:59 +0000 Packages for the release of Amarok 2.8 are available for Kubuntu 12.04 LTS, 13.04 and our Saucy development release. You can get them from the Kubuntu Backports PPA for 12.04 LTS, 12.10 and 13.04.

Bugs in the packaging should be reported to kubuntu-ppa on Launchpad. Bugs in the software to KDE.

To update, use the Software Repository Guide to add the following repository to your software sources list:

13.04, 12.10 and 12.04 LTS:

ppa:kubuntu-ppa/backports

Or, from your konsole:

sudo add-apt-repository ppa:kubuntu-ppa/backports

Thanks to Howard and Philip for their work on these packages
