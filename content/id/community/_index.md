# Komunitas

Bergabunglah dengan perjalanan Kubuntu, atau sekedar dapatkan bantuan!

## Matrix

Lihat [Halaman Matrix](/community/matrix) untuk info lebih lanjut.

## Forum Web

- [Forum Web Kubuntu](https://www.kubuntuforums.net/forum) adalah tempat terbaik untuk isu yang terkait dukungan Kubuntu.
- [Forum KDE](https://forum.kde.org/) memberi bantuan terhadap aplikasi-aplikasi dan desktop yang dipakai di Kubuntu.
- [Ask Ubuntu](https://askubuntu.com/) memiliki segudang pengetahuan dan sosok-sosok yang ramah dan siap membantu kamu dengan menjawab pertanyaan kamu.
- [Discourse Kubuntu](https://discourse.ubuntu.com/c/flavors/kubuntu/187) adalah ruang diskusi untuk Kubuntu di dalam komunitas Ubuntu keseluruhan.
